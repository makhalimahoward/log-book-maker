Log Book Maker

A fully offline industrial attachment log book workspace built for students.
Manage weekly entries, generate print-ready A4 log sheets, and export to CSV or Google Docs — all without a backend or internet connection.

Live Demo: [View Here](https://log-book-maker.pages.dev)

---

Features

- Weekly Log Management — Add, edit, and delete entries organised by week
- A4 Print Export — Generates formatted log sheets ready for university submission
- Cover Page Builder — Upload your university crest and company logo, fill in your details
- Data Export — Download as CSV or copy rich HTML tables directly into Word / Google Docs
- Dark and Light Mode — Persistent theme preference saved locally
- Fully Offline — All data stored in localStorage, no account or server needed
- Open Source — Free to use, fork, and adapt with attribution

---

Getting Started

No installation needed. Just open index.html in your browser.

git clone https://github.com/makhalimahoward/log-book-maker.git
cd log-book-maker
open index.html in your browser
